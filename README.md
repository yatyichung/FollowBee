# **FollowBee**

---

## Run the project

1) Clone project "https://github.com/yatyichung/FollowBee.git"
2) To install node_modeul run below code in command lin

    ```nodejs
    npm install
    ```

3) Install dependencies if not installed (please see Dependencies).

4) Open terminal and enter

    ```nodejs
    node index.js
    ```

---

## Dependencies

- axios 0.27.2
- express 4.8.1
- pug 3.0.2
